## Getting Started
### Clone Project

```bash
git clone https://github.com/1010code/iris-dnn-tensorflow.git
cd iris-dnn-tensorflow
```

### Installation
When that's done, install the project dependencies.
```bash
pip install -r requirements.txt
```
